Классы FileInputStream и FileOutputStream

Для чтений и записи файлов есть два класса:  FileInputStream и FileOutputStream.
FileInputStream - последовательно читает байты из файла.
FileOutputStream - последовательно записывает байты в файл.

Основные методы класса FileInputStream:

FileInputStream(String fileName) 	- конструктор, принимает в качестве параметра имя файла на диске (с указанием пути).
int read()							- читает один байт, возвращает его значение, тип значения расширяется с byte до int. Если все байты прочитаны - вернет -1.
int read(byte[] buff)				- заполняет массив байтами. Чтение выполняется до тех пор, пока массив не заполнится, либо пока все байты будут прочитаны.
int available()						- количество непрочитанных (доступных для чтения) байт.
void close()						- закрывет поток. Должен быть вызван после окончания работы с потоком для освобождения ресурсов. После вызова метода из потока больше нельзя читать данные. Помимо этого метод выполняет служебные операции, связанные с закрытием файла. 

Основные методы класса FileOutputStream:

FileOutputStream(String fileName)	- конструктор, принимает в качестве параметра имя файла на диске, в который будут записываться данные.
void write(int c)					— метод записывает один байт информации. Тип int сужается до byte, лишняя часть просто отбрасывается.
void write(byte[] buff, int from, int count) - метод записывает часть блока байт. Используется в случаях, когда есть вероятность, что блок данных был заполнен не целиком
void flush()						- записывает всю несохраненную информацию на диск. 
close()								- закрывает поток.
